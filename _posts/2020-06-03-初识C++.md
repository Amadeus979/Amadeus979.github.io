# 1.准备编译器

按照 搭建Ubuntu工作环境里的就行了 

# 2.开始学习编写Hello world

打开文本编辑器，输入以下代码：

#include <iostream>

using namespace std;

int main (){

cout<<"Hello World!"<<endl;

return 1;

}

# 3.讲解

#include <iostream> 是引入库iostream,也就是I/O输入输出流。

using namespace std; 是一个必备的语法，它导入命名空间"std"，最后以";"结尾，这是C++中必备的结束符

int main (){ 开始了一个主程序

cout<<"Hello World!<<endl; 输出"Hello World!"并换行

return 1; 也是必备的
最后以 "}" 结束main程序
